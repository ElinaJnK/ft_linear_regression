# ft_linear_regression

A first look at linear regression for Machine Learning following the subject at school 42

## Files and implemented functions
- data.csv : data file used in the program
- ft_linear_regression.py : the implemented algorithm

## Run the program

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 predict.py
python3 train.py
```
